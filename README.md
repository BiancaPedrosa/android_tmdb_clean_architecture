#### What is this?
Just a simple project to try new stuff on Android. Feel free to use it just for learning or for your next Android application.

- Kotlin
- Clean Architecture
- MVP (Model-View-Presenter)
- RxJava 2
- Dagger 2
- Retrofit
- SQLite
- Mockito
- Robolectric
- Espresso
- Travis CI

In case of any questions, feel free to open an issue.
